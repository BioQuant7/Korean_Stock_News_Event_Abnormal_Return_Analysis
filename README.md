# Korean Stock News Event Abnormal Return Analysis

국내 주요 상장기업의 종목 뉴스를 이벤트 유형별로 분류하고, 뉴스 발생 이후 단기 주가 반응을 KOSPI 대비 **초과수익률** 기준으로 분석한 Jupyter Notebook 프로젝트입니다.

## 프로젝트 개요

이 프로젝트는 네이버 금융 종목뉴스 제목을 기반으로 뉴스 이벤트를 분류한 뒤, 이벤트 발생 이후 1일, 3일, 5일 동안 해당 종목이 시장 대비 얼마나 좋은 성과를 냈는지 확인합니다.

단순 주가 수익률이 아니라 개별 종목 수익률에서 KOSPI 수익률을 차감한 초과수익률을 사용했습니다.

```text
초과수익률 = 개별 종목 수익률 - KOSPI 수익률
```

## 사용 데이터

* 주가 데이터: FinanceDataReader
* 시장 기준: KOSPI 지수
* 뉴스 데이터: 네이버 금융 종목뉴스
* 분석 대상: 국내 주요 상장기업 59개 종목
* 주가 데이터 기간: 2023-01-02 ~ 2026-04-20
* 뉴스 데이터 기간: 2025-12-02 ~ 2026-04-20
* 전체 수집 뉴스: 15,607건
* 이벤트성 뉴스: 5,377건
* 최종 분석용 일 단위 이벤트: 857건

## 분석 방법

뉴스 제목의 키워드를 기준으로 이벤트를 분류했습니다.

주요 이벤트 유형은 다음과 같습니다.

* 실적
* 수주계약
* 악재
* 목표가리포트
* 정책테마
* 투자증설
* 바이오
* 자금조달

뉴스 발생 시각은 실제 주가 반응이 나타날 거래일에 맞게 조정했습니다.

* 장중 뉴스: 해당 거래일
* 장 마감 이후 뉴스: 다음 거래일
* 주말 또는 휴장일 뉴스: 다음 거래일

이후 이벤트별 1일, 3일, 5일 초과수익률과 이벤트 전후 누적 초과수익률을 분석했습니다.

## 주요 결과

이벤트 유형별 3일 초과수익률을 비교한 결과, 뉴스 이벤트가 항상 긍정적인 주가 반응으로 이어지지는 않았습니다.

가장 양호한 이벤트는 `수주계약`이었지만, 평균 3일 초과수익률은 약 `+0.02%`로 거의 중립에 가까웠습니다.

반대로 다음 이벤트들은 단기적으로 시장 대비 부진한 흐름을 보였습니다.

* 바이오
* 악재
* 투자증설
* 정책테마

특히 `바이오` 이벤트는 평균 3일 초과수익률이 약 `-2.34%`, 5일 초과수익률이 약 `-4.12%`로 가장 부진했습니다.

이벤트 스터디 결과에서도 뉴스 발생 이후 시간이 지날수록 누적 초과수익률이 낮아지는 경향이 확인되었습니다.

## 모델링 실험

뉴스 이벤트 정보와 직전 주가 흐름을 사용해 3일 초과수익률이 상대적으로 좋은 이벤트를 분류하는 간단한 모델링 실험도 수행했습니다.

사용한 모델은 다음과 같습니다.

* DummyClassifier
* LogisticRegression
* RandomForestClassifier
* GradientBoostingClassifier

가장 높은 ROC-AUC는 GradientBoosting 모델에서 나왔으며, 주요 변수는 다음과 같았습니다.

1. 직전 5일 수익률
2. 5일 거래량 비율
3. 직전 1일 수익률
4. 제목 방향성 점수
5. 수주계약 이벤트 여부

모델이 선택한 상위 20% 이벤트는 테스트 전체보다 평균 3일 초과수익률이 덜 부진했고, 양수 비율도 높았습니다. 다만 표본 수가 많지 않아 실전 전략으로 일반화하기에는 한계가 있습니다.
